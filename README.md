# The Bat Pavilion 🏏

A premium, interactive web application featuring handcrafted Grade 1 English Willow cricket bats and an educational ePortfolio demonstrating web development concepts.

This project is built using a clean structure with separated concerns (HTML structure, CSS styles, and JavaScript logic).

## 📁 File Structure

The project has been refactored from a single monolithic file into a clean, modular structure:

*   **`index.html`**: The main structure containing the navigation, hero section, premium bat collection catalog, custom order form, and the ePortfolio.
*   **`style.css`**: The custom styling rules, layouts, color theme variables, animations, and responsive grids.
*   **`script.js`**: The interactive components including search filters, form handling, mock APIs, theme switchers, and a JavaScript playground.

---

## 🚀 Key Features

### 1. Premium Cricket Bats Showcase
*   A grid layout showcasing top-tier cricket bats (e.g., MRF Genius Grand, CEAT Hitman Pro, Gray-Nicolls Legend).
*   **Live Search Filter**: An instantaneous query filtering system using dynamic DOM element selection as you type.

### 2. Custom Order Form
*   An elegant inputs panel designed with Bootstrap and custom theme styles.
*   **Client-Side Validation & Feedback**: Prevents default postbacks, performs dynamic status validation, displays transient success indicators, and resets inputs programmatically.

### 3. Interactive ePortfolio Panels
*   **API Simulation**: Click to simulate fetch requests and rendering dynamic database results.
*   **CSS Color Theme Switcher**: Instantly preview and apply various brand styles.
*   **JS Playground**: An in-browser scratchpad to write, execute (`eval`), and read results of basic JS operations.

---

## 🛠️ Built With

*   **HTML5**: Semantic markup for proper structure, SEO, and accessibility.
*   **Bootstrap 5 CSS**: Used for structural layout grids, form styling, alerts, and navigation menus.
*   **Bootstrap Icons**: Used throughout the interface.
*   **Google Fonts**: Playfair Display (Serif) for headings and DM Sans (Sans-serif) for body copy.
*   **Vanilla CSS**: Custom branding variables, animations, and micro-interactions.
*   **Vanilla JavaScript (ES6+)**: Custom dynamic features and playground runner.
